## Базовые Статусы И Формулы

| Статус | Формула из таблицы |
| --- | --- |
| Кровотечение X (Y) | Y стаков кровотечения, X/10 за тик в течение 10 секунд |
| Горение X (Y) | Y стаков горения, X/8 за тик в течение 4 секунд |
| Яд X (Y) | Y стаков яда, X/15 за тик в течение 15 секунд, распространяется при смерти |
| Рок X (Y) | Y стаков рока, X урона, X/4 AoE-урона при смерти или срабатывании |
| Замедление X | Снижает скорость передвижения на X * 1% |
| Ошеломление X | Повышает шанс крита по цели на X * 1% |
| Хрупкость X | Повышает базовый урон по цели на X * 1 |
| Раскрытие X | Повышает базовый урон по цели до X * 4% |

## Дебаффы Получаемого Урона

| Статус | Эффект |
| --- | --- |
| Уязвимость X | Получаемый урон + X * 1% |
| Слабость X | Получаемый DoT-урон и урон рока + X * 1.2% |
| Раскол X | Броня - X * 1 |
| Дезориентация X | Получаемый критический урон + X * 1% |
| Разлад X | Получаемый урон + X * Random(-0.5%~2.5%) |
| Отвлечение X | Шанс промаха + X * 0.5% |
| Ослабление X | Наносимый урон - X * 1% |
| Увечье X | Область атаки - X * 0.5% |

## Модификаторы Урона

| Модификатор | Эффект |
| --- | --- |
| Пробивание | Урон +20% за каждого поражённого врага |
| Жестокость | Урон до +50% по ближайшим врагам |
| Точность | Урон до +100% по дальним врагам |
| Мощное попадание | Урон до +50% по врагам рядом с центром AoE |
| Сокрушение | Критический удар наносит 3x урон |
| Хаотичность | Имеет шанс `base cooldown * 20%` активировать другой навык |

## Важные Синергии

| Синергия | Условие |
| --- | --- |
| Бафф A -> B | Нужно 2 источника A |
| Цепочка навыков типа A -> B | Нужно 2 навыка типа A; навыки с КД больше 8 секунд не считаются |
| Бафф типа A к не-A | Нужно 2 навыка типа A и 2 навыка без типа A |
| Синергия дебаффа A -> B | Нужно 2 источника A |
| Шанс наложить A при попадании | Нужен 1 источник A |
| Шанс наложить A при крите | Нужно 2 источника A |
| Мгновенный урон при наложении A | Нужно 3 источника A |
| Увеличение урона A | Нужно 3 источника A |
| Увеличение частоты тиков A | Нужно 3 источника A |

## Капы Стаков

Формула из таблицы:

```text
Cap(X) = X / (1 + max(X - soft, 0) / hard)
```

Для баффов и большинства дебаффов:

```text
Soft cap = 800
Hard cap = 1000
```

Для некоторых исключений по дебаффам:

```text
Soft cap = 160
Hard cap = 200
```

## Редкость

Таблица указывает, что сначала бросается редкость, а затем выбирается возможная карта внутри этой редкости.

Редкости:

```text
Legendary -> Легендарная
Epic      -> Эпическая
Rare      -> Редкая
Uncommon  -> Необычная
Common    -> Обычная
```
