# Масштабирование Сложности По Времени

Этот документ описывает, как меняется сложность игры на протяжении 30-минутного забега.

---

## Общий Принцип

Нормализованное время `t = currentTime / 1800`, от 0 до 1.

Кривая HP врагов (медленная плавная):
```text
timeCurveHP = 1 + t^1.35 * 1.45
```

Кривая урона врагов (более медленная):
```text
timeCurveDMG = 1 + t^1.15 * 0.75
```

Сложность растёт в первую очередь через:
1. Плотность врагов (больше одновременно).
2. Состав волн (новые типы, элитки).
3. Комбинирование паттернов (дальник + рашер одновременно).
4. HP и урон врагов (вторично).

---

## Фаза 1: Освоение (0–5 минут, t: 0–0.17)

**Цель:** Игрок осваивает управление и навыки. Быстрый первый уровень.

| Параметр | Значение |
| --- | --- |
| HP врагов | ×1.00 → ×1.14 |
| Урон врагов | ×1.00 → ×1.10 |
| Скорость врагов | ×1.00 |
| Spawn Rate | 0.9 → 1.2 врагов/сек |
| Pack Size | 2–3 |
| Max Alive | 40–55 |
| XP множитель | ×1.00 |

**Состав волн:**
- Enemy_Basic: вес 1.0
- Все остальные: вес 0

**Что происходит:**
- Только рядовые враги.
- Игрок набирает первые 3–4 уровня.
- TTK обычного врага: 0.3–0.5 сек.

**Признаки нормальной фазы:**
- Игрок не теряет более 30% HP суммарно за всю фазу.
- Первый уровень набирается за 30–45 секунд.

## Фаза 2: Первое Давление (5–10 минут, t: 0.17–0.33)

**Цель:** Первые нестандартные угрозы. Нужен первый AoE навык.

| Параметр | Значение |
| --- | --- |
| HP врагов | ×1.14 → ×1.38 |
| Урон врагов | ×1.10 → ×1.21 |
| Скорость врагов | ×1.00 → ×1.02 |
| Spawn Rate | 1.2 → 1.8 врагов/сек |
| Pack Size | 3–4 |
| Max Alive | 55–75 |
| XP множитель | ×1.05 |

**Состав волн:**
- Enemy_Basic: вес 0.85 → 0.6
- Enemy_Fast: вес 0.0 → 0.35
- Enemy_Tank: вес 0.0 → 0.20
- Enemy_PoisonSpitter: вес 0.0 → 0.20
- Enemy_FrostShaman: вес 0.0 → 0.15

**Burst Events:**
- Минута 5.5: 5 PoisonSpitter одновременно
- Минута 7: первые 3 Tank одновременно (bulkwave)

**Что происходит:**
- Появляются быстрые враги и первые дальники.
- Нужно начать думать о позиции.
- Игрок должен иметь 4–6 уровней.

| Spawn Rate | 1.8 → 2.5 врагов/сек |
| Pack Size | 4–5 |
| Max Alive | 75–100 |
| XP множитель | ×1.10 |

**Состав волн:**
- Enemy_Basic: вес 0.60 → 0.40
- Enemy_Fast: вес 0.35 → 0.55
- Enemy_Tank: вес 0.20 → 0.40
- Enemy_PoisonSpitter: вес 0.20 → 0.45

- Начинается реальное позиционирование.

| XP множитель | ×1.15 |

**Состав волн:**
- Enemy_Basic: вес 0.40 → 0.25
- Enemy_Fast: вес 0.55 → 0.60
- Enemy_Tank: вес 0.40 → 0.55

## Правила Баланса Масштабирования

1. **HP растёт быстрее урона.** В конце HP ×2.45, урон ×1.75.
2. **Скорость растёт медленно.** Максимум ×1.25. Нет бесконечного ускорения.
3. **Новые типы вводятся постепенно.** Никогда не выбрасывай сразу все типы в начале.
4. **Burst Events для элиток и боссов — не в timeline, а через отдельные события.**
5. **Скорость спавна не должна создавать больше, чем maxAlive.** SpawnSystem это контролирует.
6. **Боссы имеют отдельные настройки HP/урона.** Не просто большой множитель.
