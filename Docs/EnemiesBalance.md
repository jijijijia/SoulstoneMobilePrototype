# Баланс Врагов

Этот документ содержит данные для всех типов врагов: обычные, элитные и боссы.

---

## Принципы

- Враги должны иметь чёткие визуальные роли: по внешнему виду и поведению понятно, чего ждать.
- Не делай слишком жирных врагов. HP обычного врага в начале — несколько ударов навыком.
- Сложность через разнообразие и плотность, а не через ваншот.
- Все опасные атаки должны иметь телеграф.
- Враги появляются на окружности вокруг игрока, не телепортируются внутрь.

## Боссы (3 типа)

Боссы появляются через BossManager. Используй TimedBossEvent в SpawnTimeline.

| --- | ---: |
| Health | 45 × 35 = 1575 |
| MoveSpeed | 2.0 |
| Damage | 8 × 2.5 = 20 |
| Armor | 140 |
| ExperienceReward | 50 |
| Появляется | Минута 10 (t=0.33) |

**Категория:** Boss  
**Фаза 1 (100–50% HP):** Медленный, использует GroundSlam и DashStrike.

**Ключевой паттерн:** Периодически делает BossCharge с телеграфом 1.5с — нужно уклоняться.

## Правила Спавна

1. Враги появляются на кольце радиусом ~18м вокруг игрока.
2. Обычные враги появляются постоянно. Элитки и боссы — через BurstEvent.
3. Новые типы врагов разблокируются через `unlockPlayerLevel` или через время.
4. SpawnWeight по времени настраивается в SpawnTimelineData через enemyWeightTracks.
5. Максимальное количество живых врагов ограничено maxAlive — не перегружай.
