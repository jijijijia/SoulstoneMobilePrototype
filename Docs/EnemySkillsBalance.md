# Баланс Навыков Врагов

Этот документ описывает все атаки и способности врагов с балансными параметрами.

---

## Принципы

- Все опасные атаки ДОЛЖНЫ иметь телеграф: визуальный маркер, задержку или анимацию.
- Игрок должен иметь возможность уклониться от любой атаки.
- Комбинирование нескольких паттернов повышает сложность без нечестности.
- Не делай атаки, которые наносят 80%+ HP за одно попадание.
- Не делай атаки без телеграфа с высоким уроном.

---

## Как Атаки Работают В Коде

Атаки врагов настраиваются через `EnemyAbilityConfig` и `EnemyData`.

Текущие типы атак в коде:
- `MeleeContact` — удар при контакте
- `Projectile` — снаряд
- `AreaPulse` — AoE вокруг себя

Для новых паттернов (DashStrike, DelayedImpact, PoisonPool, SummonMinions, AllyBuff, SlowField, DeathExplosion, BossCharge) потребуется расширение `EnemyAbilityConfig` или создание новых компонентов.

## Навык 1: MeleeContact (Простой Удар)

| Параметр | Значение |
| --- | --- |
| Тип | Melee, Contact |
| Базовый урон | = Enemy.Damage |
| Кулдаун | 0.8–1.5 сек |
| Дальность | PreferredDistance (1.0–2.0м) |
| Задержка перед ударом | 0 |
| Телеграф | Нет (или короткая анимация атаки) |
| Масштабирование | timeCurveDMG |
| Используется | Basic, Fast, Tank, Warhound |

**Описание:** Стандартный удар при касании. Базовая атака большинства врагов. Наносится при достижении preferred distance.

**Ограничения:** Интервал атаки не ниже 0.8с. Не должен наносить более 30% HP за удар в начале.

## Навык 2: DashStrike (Рывок)

| Параметр | Значение |
| --- | --- |
| Тип | Dash, Melee |
| Базовый урон | Enemy.Damage × 1.5 |
| Кулдаун | 4.0–5.0 сек |
| Дальность рывка | 4.0–6.0м |
| Задержка перед рывком | 0.6–0.8 сек |
| Телеграф | Враг светится/краснеет + частицы направления |
| Масштабирование | timeCurveDMG × 1.3 |
| Используется | Rusher, Elite, Champion, Boss |

**Описание:** Враг разгоняется и бросается к игроку. Наносит повышенный урон. Телеграф — мигание и направленный эффект частиц.

**Ограничения:** Не более 1 рывка в 4 секунды. Рывок должен промахнуться если игрок использовал Dash.

**Масштабирование по времени:**
| Минута | КД | Урон |
| --- | ---: | ---: |
| 0 | — | — |
| 10 | 4.0 | ×1.5 урона |
| 20 | 3.5 | ×1.5 урона × 1.45 |
| 30 | 3.0 | ×1.5 урона × 1.75 |

**Ограничения:** Радиус маркера должен точно совпадать с зоной поражения. Не спамить чаще 1 раза в 3 секунды.

| Параметр | Значение |
| --- | --- |

| Кулдаун | 6.0–8.0 сек |
| Дальность рывка | 7.0–10.0м |

| Телеграф | Яркая линия направления + нарастающий эффект + звук |

**Ограничения:** Всегда 1.5 сек телеграфа. Никогда не ниже 0.5 сек. При движении игрока босс должен корректировать направление только первые 0.5с.

## Честность Атак: Правила

1. **Любой AoE с уроном >25% HP обязан иметь ≥0.5 сек телеграфа.**
2. **Рывки не должны использоваться чаще 1 раза в 4 секунды.**
3. **Замедление не должно накапливаться до полной остановки игрока.**
4. **Взрыв при смерти обязан иметь ≥0.3 сек мигания.**
5. **Призыв союзников не должен суммарно увеличивать количество врагов более чем на 30% от текущего числа.**
6. **Боссовый рывок всегда 1.5 сек телеграфа.**
7. **Дальние атаки летят медленно — их можно избежать движением.**
