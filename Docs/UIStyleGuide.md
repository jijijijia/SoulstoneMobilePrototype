# UI Style Guide — SoulstoneMobilePrototype

Версия: 1.0 | Дата: 2026-05-05 | Платформа: Android (мобильный)

---

## Цветовая палитра

| Назначение | Hex | RGB Unity (0–1) | Применение |
|---|---|---|---|
| Panel background | `#1C1F25` | (0.11, 0.12, 0.145) | Фон панелей, боковое меню |
| Card background | `#2B2E36` | (0.17, 0.18, 0.21) | Карточки персонажей, оружия, карт |
| Accent / кнопка | `#DCAE52` | (0.86, 0.68, 0.32) | CTA-кнопки, логотип, иконки акцентов |
| Текст основной | `#FFFFFF` | (1, 1, 1) | Заголовки, HP, время |
| Текст вторичный | `#B8C0CC` | (0.72, 0.75, 0.80) | Подзаголовки, подсказки |
| XP / прогресс | `#8CC8FF` | (0.55, 0.78, 1.0) | Текст опыта, XP-слайдер |
| HP нормальный | `#FFFFFF` | (1, 1, 1) | HP-текст при HP ≥ 30% |
| HP опасность | `#FF4545` | (1, 0.27, 0.27) | HP-текст при HP < 30% |
| Cooldown fill | `#F2B838` | (0.95, 0.72, 0.22) | Заполнение кулдауна навыка |
| Slot background | `#0A0B0E` | (0.04, 0.045, 0.055) | Фон слотов кулдауна |
| Danger overlay | `#FF4444` с alpha 0.12 | — | Полупрозрачный оверлей при урону |

### Правила использования цветов

- Акцентный цвет (`#DCAE52`) — только для интерактивных элементов и важных меток. Не применять к фоновым поверхностям.
- Все панели должны иметь alpha ≤ 0.94 чтобы не перекрывать 3D-фон полностью.
- HP-текст переключается программно через `DDHUDManager.HandleHealthChanged` при пороге < 30%.
- Не использовать чисто белый фон — только тёмные поверхности (тёмная тема).

## Иерархия шрифтов (TextMeshPro)

Все текстовые элементы используют **TextMeshProUGUI**. Legacy `UnityEngine.UI.Text` — не использовать.

| Роль | Размер | Стиль | Применение |
|---|---|---|---|
| Логотип | 30 pt | Bold | Название игры в левом меню |
| Заголовок панели | 26–28 pt | Bold | Название экрана, имя персонажа |
| Кнопка основная | 22 pt | Normal | Текст CTA-кнопок |
| Текст карточки (title) | 22 pt | Normal | Название карточки |
| Текст карточки (body) | 16–20 pt | Normal | Описание в карточке |
| HUD основной | 28 pt | Normal | HP, Level, XP |
| HUD вторичный | 24 pt | Normal | Убийства, время |
| Cooldown Name | 13 pt | Bold | Название навыка в слоте |
| Cooldown Rank/CD | 12 pt | Normal/Bold | Ранг и время в слоте |
| Подсказка / hint | 24 pt | Normal | Подсказки в меню |
| Настройки (label) | 24 pt | Bold | Подпись к слайдерам |

### Правила шрифтов

- Минимальный размер для читаемости на мобильном — **12 pt** (в слотах кулдауна).
- На мобильных устройствах рекомендуется не опускаться ниже **14 pt** для основного текста.
- Все кнопки — `TextAlignmentOptions.Center`, описания — `TopLeft` или `Left`.
- `TextWrappingModes.Normal` + `TextOverflowModes.Ellipsis` — стандарт для всех TMP-текстов.

## Состояния кнопок

Все кнопки используют `Selectable.Transition.ColorTint`.

| Состояние | Цвет | Применение |
|---|---|---|
| Normal | Card background `#2B2E36` | Обычное состояние |
| Highlighted | Слегка светлее `#3A3E4A` или тёмно-золотой `#47400F` | Наведение мыши |
| Pressed | 82% от Normal brightness | Нажатие |
| Disabled | 50% alpha от Normal | Недоступна |
| Selected (карточка) | Accent `#DCAE52` | Выбранная карта/персонаж/оружие |

### Scale punch анимация

При нажатии любой `MenuCardViewBase` и кнопок level-up:
- Scale: `1.0 → 0.93 → 1.0` за 0.1 сек
- Компонент: `ButtonPunchAnimation` (автоматически добавляется)
- Использует `Time.unscaledDeltaTime` — работает при паузе

## Safe Area

Safe area обязателен для всех сцен.

- Компонент `SafeAreaFitter` вешается на **корневой RectTransform** UI в каждой сцене.
- В `MainMenu.unity` — на `MainMenu_Backdrop`.
- В `main.unity` — на `HUDPanel`.
- `SafeAreaFitter` автоматически подстраивает `anchorMin/anchorMax` под `Screen.safeArea` и обновляется при изменении ориентации.

## Fade-in переходы

При переключении экранов меню:

- Целевая панель появляется с `CanvasGroup.alpha = 0` и плавно переходит к `alpha = 1` за **0.12 сек**.
- Скрытые панели деактивируются мгновенно (`SetActive(false)`).
- Использует `Time.unscaledDeltaTime`.
- Реализовано в `MainMenuManager.ShowScreen()` → `FadeInPanel()`.

## Правила русского текста

- Весь UI-текст — на русском языке.
- Технические состояния: `"Бой"`, `"Пауза"`, `"Победа!"`, `"Поражение"`, `"Новый уровень"`.
- Метки HUD: `"HP:"`, `"Убийств:"`, `"Уровень"`.
- Статы: `"Время:"`, `"Уровень:"`, `"Аффиксы:"`, `"Навыки:"`.
- Кнопки действий: `"В БОЙ"`, `"ВЫБРАТЬ"`, `"НАЗАД"` — капслок.
- Кулдаун: `"Готов"` (не "Ready").
- Настройки: `"Качество: Среднее"`, `"Язык: Русский"` — с текущим значением.
- Не использовать машинный перевод — всегда проверять смысл.

## Canvas настройки

- Canvas Scaler: `Scale With Screen Size`
- Reference resolution: `1920 × 1080`
- Match Width Or Height: `0.5`
- Render mode: `Screen Space – Overlay`

## Анимации level-up карточек

При наведении на кнопку выбора навыка:
- `Outline.effectColor` плавно переходит от `alpha=0` к `rgba(0.86, 0.68, 0.32, 0.85)` за 0.1 сек.
- При уходе курсора — обратный переход.
- Компонент: `LevelUpButtonHighlight` (добавляется динамически в `DDLevelUpManager`).
