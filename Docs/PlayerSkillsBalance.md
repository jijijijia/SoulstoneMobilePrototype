# Баланс Активных Навыков Игрока

Этот документ содержит полные балансовые данные для всех активных навыков проекта.

---

## Принципы Навыков

- Навыки должны иметь чёткие роли: антитолпа, антиэлита, контроль, безопасность, масштабируемость.
- Суммарный DPS всех навыков должен расти с уровнями билда примерно x2.5–4.0 к 25-му уровню.
- Быстрые частые навыки = меньше урона за удар. Мощные редкие = больше урона, долгий КД.
- Не делай навыки, которые покрывают весь экран с 1-го уровня.
- AoE-навыки слабее против боссов. Projectile сильнее против боссов.

---

## Таблица Навыков

### Условные Обозначения

- **Роль:** AT = антитолпа, AE = антиэлита, CC = контроль, S = безопасность, GS = масштабируется поздно
- **КД** — базовый кулдаун в секундах
- **DMG** — базовый урон
- **Радиус** — в метрах (Unity units)
- **Макс Ранг** — максимальный уровень навыка

## Правила Баланса Навыков

1. **Максимум 6 активных навыков** у игрока одновременно.
2. Навык не должен покрывать весь экран с Ранга 1.
3. Арена в 5–10 метров радиуса — нормально для AreaPulse Ранга 5.
4. Сильные навыки (Armageddon) должны оставаться ограниченными длинным КД.
5. Не делай NavSkills с DPS выше 150 единиц/с на Ранге 5.
6. Контроль (замедление, оглушение) не должен быть постоянным. Максимум 40% uptime.
